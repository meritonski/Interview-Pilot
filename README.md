## Privacy Policy

**Senast uppdaterad:** 2026-01-09

Interview Pilot respekterar din integritet.

### 1) Data vi samlar in
Som standard samlar vi **inte** in någon personlig data, och vi använder **ingen** spårning.

Appen kan lagra din data lokalt på din enhet, t.ex.:
- Jobb/ansökningar du lägger in
- Förberedelsesteg du skapar
- Intervjudatum och påminnelseinställningar

### 2) iCloud (valfritt)
Om du aktiverar iCloud kan din data synkas via **Apple iCloud/CloudKit** mellan dina egna enheter.

- Data kopplas till ditt Apple-ID enligt Apples regler.
- Vi (utvecklaren) kan inte se din data i iCloud.

### 3) Notiser (valfritt)
Om du aktiverar notiser används de endast för att visa påminnelser om intervjuer/uppgifter som du själv har skapat.

- Notiserna skapas lokalt på din enhet.
- Vi skickar inte notiser från en server.

### 4) Tredjepart
Appen använder inga tredjeparts-annonser, inga trackers och inga externa analytics som samlar personlig data.

### 5) Delning av data
Vi säljer, delar eller hyr **aldrig** ut din data.

### 6) Datasäkerhet
Din data lagras lokalt på enheten och (om du väljer det) via Apples iCloud. Apple hanterar kryptering och konto-säkerhet för iCloud.

### 7) Radering av data
Du kan radera din data genom att:
- radera objekt i appen, eller
- avinstallera appen (tar bort lokal data)
- om iCloud är aktivt: hantera/ta bort appens data via iCloud-inställningar (Apple ID → iCloud → Hantera lagring)

### 8) Barn
Appen är inte riktad till barn och vi samlar inte medvetet in data från barn.

### 9) Kontakt
Har du frågor om integritet:
- **E-post:** support@interviewpilot.app
